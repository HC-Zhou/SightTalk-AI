# AI Vision Dialogue Assistant Demo Script

## Demo Goal

展示应用可以打开摄像头和麦克风，通过 WebSocket 将媒体事件发给后端，完成一轮视觉对话，尝试播放 TTS 响应，并显示成本控制指标。

## Steps

1. 打开 `http://127.0.0.1:5173`。
2. 点击 `开始`。
3. 授权浏览器访问摄像头和麦克风。
4. 将一个物体放在摄像头前。
5. 提问：`我手里拿的是什么？`
6. 点击 `我说完了`。
7. 观察 UI 状态：
   - 摄像头预览可见
   - 麦克风状态为 recording
   - 用户转写出现
   - 助手文字回答出现
   - 浏览器尝试播放 TTS 音频
   - 成本面板显示 ASR = 1、视觉问答 = 1、TTS = 1
8. 说明成本策略：
   - 前端抽样帧，而不是上传原始视频流
   - 后端只保留最近帧
   - 每轮只触发一次 ASR、一次多模态问答和一次 TTS
   - mock 适配器让本地演示可重复且无付费调用
